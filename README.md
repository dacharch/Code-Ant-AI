## Getting Started



### Prerequisites

Ensure you have the following installed:

- **Node.js** (v16 or higher): [Download Node.js](https://nodejs.org/)
- **npm** (comes with Node.js) or **yarn**

### Installation

1. **Clone the repository:**

   ```bash

2. **Change the Directory **

3. **Install the Files **
